About the Program

This is a simple Java program that allows the user to:

Add student details

Store multiple students using an ArrayList

Display all student records

Automatically calculate grade based on marks

The program uses concepts like classes, inheritance, methods, and basic user input.

How to Run

Save the code in a file named StudentRecordSystem.java

Compile the file:

javac StudentRecordSystem.java


Run the program:

java StudentRecordSystem

Features

Add student with roll number, name, course, and marks

Validate marks (0–100)

Calculate grade (A, B, C, D)

Display all students

Simple menu-driven program

Classes Used

Person – Base class

Student – Inherits Person and stores student details

StudentRecordSystem – Main class
